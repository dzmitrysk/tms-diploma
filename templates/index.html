API rules:
To receive information about employees send GET request:

GET 'http://127.0.0.1:8000/api/employees'

Response returns JSON with description of all employees.
To receive information about equipments send GET request:

GET 'http://127.0.0.1:8000/api/equipments'

Response returns JSON with description of all equipments.
To give an equipment to employee send POST request:

POST 'http://127.0.0.1:8000/api/give/<equipment_id>'
-- form 'employee=<employee_id>'

Response returns JSON with description of equipment was given.
